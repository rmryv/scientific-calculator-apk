# Scientific Calculator - Android App

A full-featured scientific calculator built with Kotlin + Jetpack Compose.

Includes all standard scientific functions:
- Basic arithmetic + - × ÷ ^ (power)
- Trigonometry: sin/cos/tan + asin/acos/atan (DEG/RAD mode)
- Logarithms: log10, ln
- Roots & powers: sqrt, x², x^y, 10^x, e^x
- Constants: π, e
- Other: 1/x, %, |x|, n! (factorial), ±
- Memory: MC, MR, M+, M-
- Backspace, clear, error handling

## Quick Start

### Build Debug APK via GitHub Actions (easiest)

1. The workflow runs automatically.
2. Go to **Actions** tab.
3. Download the **Calculator-debug-apk** artifact.

Install the APK on your phone (enable 'Install from unknown sources' or use ADB).
